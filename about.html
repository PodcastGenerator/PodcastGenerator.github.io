About Podcast Generator

Podcast Generator (PG) is an open source Content Management System written in PHP and specifically designed for podcast publishing. It provides the user with the tools to easily manage all of the aspects related to the publication of a podcast, from the upload of episodes to its submission to the iTunes or Spotify Store. There is no need for a MySQL Database. It is free open source software which means anyone can contribute code or translate the project to any language.
